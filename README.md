sbt sample
==========

Overview
---------

A sample scala project which uses sbt as a build tool.

Build
---------

```
  sbt compile
```

Run
---------

```
  sbt run
```

"Hi" is printed in standard output

Test (ScalaTest)
---------

```
  sbt test
```
